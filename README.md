Прошивка для устройства на базе платы Arduino Nano для точного дозирования клея с помощью пневматики и электроклапана.
Время срабатывания задается поворотным энкодером в пределах 100-4000 миллисекунд. Запуск с помощью кнопки без фиксации со щелчком. 
Для отображения времени используется либо LCD 1602 с i2c либо набор из четырех семисегментных индикаторов TM1637 с интерфейсом i2c.
